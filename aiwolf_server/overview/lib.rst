===================
人狼知能とは
===================


概要
====

人狼知能プロジェクトとは不完全情報コミュニケーションゲーム「汝は人狼なりや」をプレイできる人工知能を作ることで

対人コミュニケーションを持つ人工知能を作ることを目的とするプロジェクトである。

このドキュメントでは実際の人狼ではなく、人狼知能におけるサーバーの実装や人狼知能プロトコル、ルールに関しての記述を行う。

また、以下の文章はaiwolf.orgのレポジトリのソースコードを参考にして記述したものである。
