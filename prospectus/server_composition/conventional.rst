従来のサーバー
=============

現在(2017年7月時点)の人狼知能の公式ではTCP/IP通信のサーバーとJavaのクラスを直接読み込むサーバーが用意されている。
また非公式ではあるが、それら２つの機能を合わせたハイブリッドサーバーと呼ばれるものも使われている。
公式の大会ではTCP/IP通信のサーバーが使われているはずである。
